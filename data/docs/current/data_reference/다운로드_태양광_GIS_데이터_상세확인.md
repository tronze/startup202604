## 1-1. 누가, 어떻게 만든 데이터인가

이 자료는 한국 기관이 현장에서 모든 격자를 직접 측정한 원자료가 아니다. Solargis가 만든 글로벌 태양광 자원 모델의 산출물이고, Global Solar Atlas를 통해 공개된 데이터다.

역할 구분:

| 주체 | 역할 |
| --- | --- |
| Solargis | 태양광 자원/발전잠재량 모델 개발 및 데이터 산출 |
| Global Solar Atlas | 공개 플랫폼 |
| ESMAP | World Bank가 관리하는 다자기금, Global Solar Atlas 사업 funding |
| World Bank | ESMAP 관리 및 공개 데이터 플랫폼 제공 |

생성 방식:

- `GHI`, `DNI`, `DIF`, `GTI` 같은 일사량 layer는 Solargis 알고리즘으로 산출된다.
- 메타데이터상 주요 입력은 위성 디지털 영상과 대기 자료다.
- 사용된 입력원으로 Meteosat, GOES, MTSAT/Himawari-8, ECMWF MACC-II/CAMS, NASA MERRA-2, NOAA GFS 등이 명시되어 있다.
- `PVOUT`은 실측 발전소 발전량이 아니라 Solargis 알고리즘으로 계산한 잠재 발전량이다.
- `PVOUT`의 주요 입력은 최적 경사면 일사량 `GTI`와 기온 `TEMP`다.

목적:

- 장기 평균 태양광 자원과 발전잠재량을 시각화/분석하기 위한 reference layer
- free-standing PV plant를 최적 경사각으로 설치했을 때의 발전잠재량 평가
- 국가/지역 단위 재생에너지 자원 mapping
- 부지 screening, 후보지 비교, 초기 수익성 추정

주의:

- 이 데이터는 2007-2018 장기평균이다.
- 특정 연도 실제 발전량이나 특정 발전소의 실측 발전량이 아니다.
- 투자 판단용으로 쓰려면 현장 음영, 지형, 패널 배치, 계통접속, 출력제어, CAPEX/OPEX를 별도로 붙여야 한다.

## 2. 들어있는 파일

연평균/장기평균 layer:

| 파일 | 의미 | 단위 | 해상도 |
| --- | --- | --- | --- |
| `PVOUT.tif` | 태양광 발전 잠재량 | kWh/kWp/day | 30 arc-sec, 약 1km |
| `GHI.tif` | 수평면 전일사량 | kWh/m²/day | 9 arc-sec, 약 250m |
| `GTI.tif` | 최적 경사면 전일사량 | kWh/m²/day | 9 arc-sec, 약 250m |
| `DNI.tif` | 직달일사량 | kWh/m²/day | 9 arc-sec, 약 250m |
| `DIF.tif` | 산란일사량 | kWh/m²/day | 9 arc-sec, 약 250m |
| `TEMP.tif` | 기온 | °C | 30 arc-sec, 약 1km |
| `OPTA.tif` | 최적 경사각 | degree | 2 arc-min, 약 4km |

월별 layer:

- `monthly/PVOUT_01.tif`부터 `monthly/PVOUT_12.tif`
- 각 월의 장기 평균 PVOUT
- 단위는 kWh/kWp/day
- 해상도는 약 1km

메타데이터:

- 각 `.tif`마다 `.xml`, `.pdf`, `.aux.xml` 메타데이터가 같이 들어 있다.

## 3. 공간/시간 상세도

공간 범위:

- 서경계: 124.0
- 동경계: 132.0
- 남경계: 33.0
- 북경계: 39.0
- 좌표계: EPSG:4326, WGS84

시간 범위:

- 2007-2018 장기평균

격자 크기:

| layer | raster 크기 | 실제 값이 있는 grid 수 | 격자 간격 |
| --- | ---: | ---: | --- |
| PVOUT | 960 x 720 | 205,427 | 0.008333° |
| GHI | 3,200 x 2,400 | 2,283,659 | 0.0025° |
| GTI | 3,200 x 2,400 | 2,283,659 | 0.0025° |
| DNI | 3,200 x 2,400 | 2,283,659 | 0.0025° |
| DIF | 3,200 x 2,400 | 2,283,659 | 0.0025° |
| TEMP | 960 x 720 | 207,693 | 0.008333° |
| OPTA | 240 x 180 | 13,290 | 0.033333° |

해석:

- `GHI`, `GTI`, `DNI`, `DIF`는 전국을 약 250m 격자로 볼 수 있다.
- `PVOUT`은 전국을 약 1km 격자로 볼 수 있다.
- 수익 추정에는 `PVOUT`을 바로 쓰는 것이 가장 빠르다.
- 부지/필지 단위까지 내려가려면 이 데이터에 필지, 지형, 건물/음영, 계통 데이터를 추가해야 한다.

## 4. 값 범위

장기평균 layer 값:

| layer | min | mean | max |
| --- | ---: | ---: | ---: |
| PVOUT | 2.826 | 3.800 | 4.138 |
| GHI | 2.461 | 3.991 | 4.262 |
| GTI | 2.367 | 4.552 | 4.932 |
| DNI | 1.364 | 3.353 | 3.890 |
| DIF | 1.432 | 1.950 | 2.076 |
| TEMP | 3.700 | 12.770 | 17.100 |
| OPTA | 25.000 | 32.694 | 37.000 |

PVOUT 해석:

```text
연간 발전량(kWh/year)
= 설비용량(kWp) * PVOUT(kWh/kWp/day) * 365.25
```

한국 내 PVOUT 범위:

```text
최저: 2.826 * 365.25 = 약 1,032 kWh/kWp/year
평균: 3.800 * 365.25 = 약 1,388 kWh/kWp/year
최고: 4.138 * 365.25 = 약 1,511 kWh/kWp/year
```

즉 100kW 설비 기준으로는 대략:

```text
최저: 약 103,200 kWh/year
평균: 약 138,800 kWh/year
최고: 약 151,100 kWh/year
```

## 7. 우리가 만들 수 있는 분석 제품

이 데이터만으로 가능한 MVP:

- 전국 1km 격자별 PVOUT map
- 전국 250m 격자별 GHI/GTI/DNI/DIF map
- 특정 좌표 입력 시 예상 연간 발전량
- 100kW/500kW/1MW 시나리오별 예상 발전량
- 판매단가별 단순 매출 추정
- 월별 발전량 profile
- 시군구 평균 PVOUT ranking

추가 데이터가 있으면 가능한 고급 모델:

- 필지별 설치 가능 면적
- 건물/산지 음영 손실
- 경사/방위 보정
- 한전 접속 가능성 score
- 출력제어 risk score
- CAPEX/OPEX/IRR/NPV

## 8. 실무 판단

이 데이터는 생각보다 쓸 수 있는 수준이다.

장점:

- 전국 coverage가 있다.
- PVOUT이 있어서 발전량 추정으로 바로 연결된다.
- GHI/GTI는 250m 수준이라 시군구보다 훨씬 자세하다.
- 월별 PVOUT이 있어 계절별 수익 변동을 볼 수 있다.

한계:

- 장기평균 자료라 특정 연도 실적을 그대로 맞추는 데이터는 아니다.
- 1km/250m 격자라 필지 내부의 건물 그림자, 산 그림자, 패널 방위는 직접 반영하지 않는다.
- 계통 접속과 출력제어 손실은 전혀 반영되지 않는다.

따라서 이 데이터는 “태양광 자원 잠재량”을 보는 1차 layer로 쓰고, 우리 서비스의 차별점은 여기에 계통/출력제어/경제성 layer를 얹는 방향이 맞다.
