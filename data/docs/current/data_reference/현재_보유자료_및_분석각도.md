# 현재 보유자료 및 분석 각도

기준일: 2026-04-27  
범위: 현재 repository에 남겨둔 `data/raw/`, `data/processed/` 자료만 기준으로 정리한다.  
목적: Excel로 직접 다루기 어려운 자료를 SQL/CSV 분석 기준으로 재정리하고, 지금 당장 볼 수 있는 분석 방향을 확정한다.

## 1. 결론

현재 자료는 Excel에서 직접 작업하기보다 SQL로 올려서 보는 것이 맞다. 시간별 수요, 시간별 SMP, 일별 기상, 일별 수급, DR, 월별 재생에너지 이용률, KPX 공지 텍스트가 서로 다른 단위로 섞여 있기 때문이다.

지금 보유한 데이터로 바로 분석 가능한 주제는 다음이다.

- 전국 시간별 전력수요의 장기 패턴: 2013-2025
- SMP 가격 패턴과 0원 SMP 발생: 2020-2025
- 날씨와 전력수요의 비선형 관계: 2013-2025
- Plus DR 입찰/낙찰과 가격/수요의 관계: 2020-2021
- 예비율과 저부하/고부하 조건: 2020-2025
- 재생에너지 이용률의 월별/지역별 차이: 2018-2025
- KPX 출력제어 관련 공지의 키워드/시점 변화: 2024-11-05부터 2026-04-12까지

아직 직접 답하기 어려운 것은 실제 출력제어량(MWh), 발전기별/사업자별 손실, 변전소 단위 역조류, 고객별 수요반응 경제성이다. 이 값들은 현재 자료에 직접 수치로 들어 있지 않다.

- `supply_daily.csv`의 `min_load_mw`는 대부분 비어 있다. 최소수요 분석은 이 컬럼만 믿으면 안 되고 시간별 수요에서 직접 계산해야 한다.
- KPX 공지 detail의 `archive_html` 컬럼은 과거 crawl archive 경로를 가리킨다. archive HTML 파일은 정리 과정에서 제거했고, 현재 분석에는 CSV 안의 `body_text`를 사용한다.
- `market_statistics_sheet_index.csv`는 Excel sheet를 대량 CSV로 펼치지 않고 index만 남긴 것이다. 특정 sheet 분석이 필요하면 원본 Excel에서 해당 sheet만 추출한다.

## 3. 데이터 연결 기준

| 연결 단위 | 연결 가능한 자료 | 분석 의미 |
| --- | --- | --- |
| `date` + `hour` | 수요, SMP, DR | 시간별 가격/수요/DR 관계 |
| `date` | 시간별 수요 집계, 수급, 기상 | 일별 예비율, 날씨, 수요 관계 |
| `period` 월 | 월별 SMP, 재생에너지 이용률, 지역 에너지 사용량 | 월별/지역별 구조 비교 |
| 공지 `date` | KPX 공지, 수요/SMP/수급 event window | 공지 전후의 시장/계통 상태 비교 |
| `region` | 재생에너지 이용률, 지역 에너지 사용량 | 지역별 수급 mismatch 가설 검토 |

SQL로 볼 때는 `date`, `hour`, `period`, `region` 기준으로 join하는 것이 가장 안정적이다.

## 4. 지금 볼 수 있는 분석 각도

### A. 저부하 + 0원 SMP + 출력제어 압력

핵심 질문:

- 주말 낮 시간대에 수요가 얼마나 낮아지는가?
- SMP가 0원이 되는 시간은 어떤 요일/시간/계절에 몰리는가?
- KPX 출력제어 공지 시점과 저부하/0원 SMP가 같이 나타나는가?

사용 자료:

- `load_hourly_all_sources.csv`
- `smp_hourly.csv`
- `weather_daily.csv`
- `supply_daily.csv`
- `kpx_output_related_notice_details.csv`

현재 근거:

- SMP 시간별 데이터에서 0원 SMP는 2024년에 6시간, 2025년에 19시간 존재한다.
- 기존 processed 분석에서는 0원 SMP가 주말 낮 시간대에 집중되는 패턴이 확인됐다.
- 출력제어 관련 KPX 공지 200건이 있어 공지일 전후 event study를 할 수 있다.

한계:

- 실제 출력제어 MWh가 없으므로 “출력제어가 얼마 발생했다”는 직접 계산할 수 없다.
- 현재는 “출력제어 압력이 높아지는 조건”을 추정하는 분석이 적합하다.

### B. 수요-가격 관계와 가격 regime 변화

핵심 질문:

- 수요가 낮을수록 SMP가 낮아지는가?
- 2022-2023 가격 급등과 2024-2025 가격 하락은 수요와 어떻게 다르게 움직였는가?
- 0원 SMP는 구조적 변화의 신호인가, 일부 event인가?

사용 자료:

- `load_hourly_all_sources.csv`
- `smp_hourly.csv`
- `smp_weighted_monthly.csv`
- `weather_daily.csv`

현재 근거:

- 시간별 SMP 평균은 2020년 약 67.74, 2021년 약 92.99, 2022년 약 193.75, 2023년 약 163.27, 2024년 약 125.44, 2025년 약 110.76원/kWh 수준이다.
- 수요와 SMP는 같은 방향으로 움직이는 경향이 있지만, 연료비/시장제도 영향도 섞여 있어 단순 상관만으로 설명하면 부족하다.

분석 방식:

- 연도별 고정효과를 둔 회귀
- 시간대/요일/계절별 box plot
- 저수요 분위별 SMP 비교
- 0원 SMP event table 생성

### C. 날씨 기반 전력수요 예측

핵심 질문:

- 기온과 수요는 U-shape 관계인가?
- 폭염/한파일에 수요가 얼마나 증가하는가?
- 서울/인천/부산/대구 중 어떤 도시 기상 변수가 전국 수요 proxy로 더 잘 맞는가?

사용 자료:

- `weather_daily.csv`
- `load_hourly_all_sources.csv`

현재 근거:

- 기상 자료는 서울, 인천, 부산, 대구 각각 4,748일씩 있다.
- 기존 분석에서 기온-수요 관계는 단순 선형보다 2차식 설명력이 높았다.

분석 방식:

- 일별 평균/최대 수요와 평균기온 결합
- 냉방도일/CDD, 난방도일/HDD 생성
- 주말/공휴일 여부와 계절 고정효과 추가

### D. DR의 시장 반응성

핵심 질문:

- DR 입찰/낙찰이 있는 시간은 SMP가 높은가?
- DR 활성 시간은 최대수요/예비율과 관련이 있는가?
- 경제성 DR, 피크 DR, 미세먼지 DR이 서로 다른 조건에서 나타나는가?

사용 자료:

- `dr_hourly.csv`
- `smp_hourly.csv`
- `load_hourly_all_sources.csv`
- `supply_daily.csv`

현재 근거:

- DR 자료는 2020-2021 시간별 17,544행이다.
- 이 중 입찰/낙찰 count가 하나라도 있는 활성 시간은 11,976시간이다.
- 기존 분석에서는 DR 활성 시간대의 평균 SMP가 비활성 시간보다 높게 나타났다.

한계:

- 현재 DR 자료는 count 중심이다. MW 규모, 정산금, 참여 자원별 성과가 없으므로 경제성 분석은 제한적이다.

### E. 예비율과 계통 여유

핵심 질문:

- 예비율이 낮은 날과 높은 날의 수요/가격은 어떻게 다른가?
- 공급능력 대비 수요가 높은 날은 어떤 계절/요일에 집중되는가?
- 저부하일에는 예비율이 높고 SMP가 약한가?

사용 자료:

- `supply_daily.csv`
- `load_hourly_all_sources.csv`
- `smp_hourly.csv`

주의:

- `supply_daily.csv`의 `min_load_mw`가 대부분 비어 있으므로 최소수요는 시간별 수요에서 직접 산출해야 한다.

### F. 재생에너지 이용률과 지역 mismatch

핵심 질문:

- 지역별 태양광 이용률은 월별로 어떻게 달라지는가?
- 이용률이 높은 지역과 에너지 사용량이 큰 지역이 일치하는가?
- 제주/호남/경북 등 특정 지역에서 출력제어 관련 공지가 더 자주 언급되는가?

사용 자료:

- `renewable_utilization_monthly.csv`
- `regional_energy_usage_2024.csv`
- `kpx_output_related_notice_details.csv`

현재 근거:

- 재생에너지 이용률은 17개 지역, 2018-2025 월별 자료가 있다.
- 현재 가공된 fuel은 태양광 중심이다.
- 지역별 에너지 사용량은 2024년 월별 자료가 있다.

한계:

- 실제 발전량, 설비용량, 출력제어량이 없어서 “이용률 하락 원인이 출력제어”라고 직접 단정할 수 없다.

### G. KPX 출력제어 공지 텍스트 분석

핵심 질문:

- 출력제어 공지가 어떤 계절/기간에 집중되는가?
- 제주, 육지, 경부하기, 수급, 태양광 같은 키워드의 흐름은 어떤가?
- 공지일 전후의 수요/SMP/예비율은 일반일과 다른가?

사용 자료:

- `kpx_output_related_notice_details.csv`
- `load_hourly_all_sources.csv`
- `smp_hourly.csv`
- `supply_daily.csv`

현재 근거:

- 출력제어 관련 공지 detail은 200건이다.
- 공지 기간은 2024-11-05부터 2026-04-12까지다.
- 제목/본문 기준으로 `출력제어`, `경부하기`, `수급`, `제주`, `육지`, `태양광` 같은 키워드를 바로 집계할 수 있다.

분석 방식:

- 월별 공지 수
- 키워드별 공지 수
- 공지일 전후 3일 event window
- SMP 0원 또는 저부하 시간과의 동시성 비교

2. `analysis_hourly.csv`를 바로 믿기보다 `load_hourly_all_sources.csv`를 기준으로 SMP, 기상, 수급, DR을 다시 붙인 분석용 view를 만든다.

3. 첫 분석 리포트는 아래 5개 표/차트로 시작한다.

- 연도별 평균 수요와 평균 SMP
- 요일/시간대별 수요 heatmap
- 0원 SMP 발생 시간 목록과 전후 수요/기상/예비율
- DR 활성/비활성 시간의 SMP와 수요 비교
- 기온 구간별 평균 수요 U-shape chart

4. 출력제어 사업 가설은 “현재 직접 증명”이 아니라 “압력 조건 탐지”로 표현한다.

현재 자료만으로 방어 가능한 표현:

- 낮은 수요, 주말 낮, 높은 재생에너지 이용률, 낮은 SMP/0원 SMP, 출력제어 공지가 함께 나타나는 조건을 찾는다.
- 이 조건을 기반으로 출력제어 위험 시간대 예측, DR/ESS/수요이전 기회 탐지, 데이터 gap 보완 전략을 제안한다.

아직 방어하기 어려운 표현:

- 특정 시간에 실제 출력제어량이 얼마였다고 단정
- 특정 사업자의 손실액 산정
- 특정 지역의 계통 병목을 변전소 단위로 확정
