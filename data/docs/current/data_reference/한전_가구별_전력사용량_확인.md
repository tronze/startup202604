# 한전 가구별 전력사용량 데이터 확인

기준일: 2026-04-27

## 결론

한전에서 공개적으로 받을 수 있는 `가구 평균 전력사용량`은 개별 가구별 원자료가 아니다. 지역별, 월별, 주택용 고객 대상의 평균 사용량/평균요금 통계다.

개별 고객 또는 계량기 단위의 상세 사용량은 한전 전력데이터서비스마켓(EDS) API에 존재한다. 다만 이 데이터는 일반 파일 다운로드처럼 전체 가구 데이터를 받는 구조가 아니라, 고객번호/계기번호 기반으로 조회하거나 고객 동의 및 사업자용 API 절차가 필요한 성격이다.

## 공개 다운로드 가능한 자료

### 1. 한국전력공사_가구평균 전력사용량

출처:

- 공공데이터포털: https://www.data.go.kr/data/15039291/fileData.do
- 한전 전력데이터 개방포털: https://bigdata.kepco.co.kr/cmsmain.do?scode=S01&pcode=000171&pstate=L&redirect=Y

성격:

- 전국 시군구별 가구 평균 전력사용량
- 주택용 계약종별 고객 대상
- 기간은 2013년부터 선택 가능
- 월별 자료
- 조회는 가능하지만 다운로드에는 회원가입/로그인이 필요하다고 명시됨

제공 필드:

- 지역구분
- 대상가구수(호)
- 가구당 평균 사용량(kWh)
- 가구당 평균 전기요금(원)

해석:

- 이 자료는 “가구 A, 가구 B의 사용량”이 아니다.
- `서울 강남구 주택용 고객의 월평균 사용량` 같은 지역 평균 통계에 가깝다.
- 개인 식별, 고객번호, 계량기번호, 시간대별 load profile은 없다.

### 2. 한국전력공사_가구 평균 전력사용량 OpenAPI

출처:

- 공공데이터포털: https://www.data.go.kr/data/15101404/openapi.do

성격:

- 전력데이터개방포털 API
- JSON/XML
- 회원가입 후 사용 가능
- 공개 파일과 마찬가지로 `가구별 평균 전력사용량` API다.

해석:

- API라고 해서 개별 가구 raw data가 나오는 것은 아니다.
- 지역/기간 조건으로 평균 통계를 조회하는 용도에 가깝다.

## 더 상세한 고객 단위 자료

### 한전 EDS OPEN API 서비스 목록

현재 repository에 이미 보유한 파일:

- `data/processed/kepco_api_catalog_20250630.csv`

공식 출처:

- 공공데이터포털: https://www.data.go.kr/data/15131481/fileData.do

확인된 관련 API:

| service_id | API 서비스명 | 제공내용 |
| ---: | --- | --- |
| 1 | 고객정보 | 요청고객의 고객기본사항 |
| 2 | 고객번호, 계기번호 | 고객번호, 계기번호 조회 |
| 3 | 청구정보 | 요청고객의 전기요금 청구정보 |
| 5 | 15분단위 전력소비 데이터 | 요청고객의 요청일자시분에 대한 15분단위 1개 데이터 |
| 6 | 일단위 전력소비 데이터 | 요청고객의 요청일자에 대한 15분단위 하루치 96개 데이터 |
| 7 | 특정고객의 특정시간 전력소비 데이터 | 고객의 요청기간에 대한 전력소비 데이터, 최대 조회기간 3개월 |
| 8 | 전체고객의 전력소비 데이터 | 전체고객의 요청일시 96개 데이터 |
| 9 | 일단위 전력소비 데이터(DR) | 요청고객의 요청일자에 대한 15분단위 하루치 96개 데이터 |
| 10 | 15분단위 전력소비 데이터(DR) | 요청고객의 요청일자시분에 대한 15분단위 1개 데이터 |
| 501 | 15분단위 저압 전력소비 데이터(DR) | 요청고객의 요청일자시분에 대한 15분단위 1개 데이터 |

해석:

- 여기에는 단순 월 사용량보다 훨씬 상세한 15분 단위 사용량이 있다.
- 하루치는 96개 point다.
- 특정 고객에 대해 최대 3개월 범위 조회가 가능한 API도 있다.
- 고객번호/계기번호와 고객 동의 또는 EDS 사업자 권한이 필요할 가능성이 높다.

## 구분 정리

| 원하는 데이터 | 다운로드 가능성 | 상세도 | 현실적 접근 |
| --- | --- | --- | --- |
| 지역별 가구 평균 사용량 | 가능 | 월별, 시도/시군구 평균 | 한전 개방포털 로그인 후 다운로드 또는 API |
| 지역별 대상가구수/평균요금 | 가능 | 월별, 지역 평균 | 같은 자료에서 제공 |
| 개별 가구 월별 사용량 | 공개 다운로드는 어려움 | 고객/계량기 단위 | 고객 동의, 한전 EDS, 기관 협조 필요 |
| 개별 가구 15분 사용량 | EDS API에 항목 존재 | 15분 단위 96개/day | 고객번호/계기번호, 고객 동의, EDS 사업자 API 필요 |
| 전체 가구 raw data bulk | 일반 공개자료로는 확인 안 됨 | 개인정보 위험 높음 | 비식별/가명 결합, 데이터안심구역, 협약 필요 |

## 분석 관점

현재 우리가 바로 받을 수 있는 공개 자료로 가능한 분석:

- 지역별 주택용 월평균 사용량 비교
- 여름/겨울 가구당 평균 사용량 계절성
- 평균 전기요금과 평균 사용량의 지역별 차이
- 현재 보유한 `regional_energy_usage_2024.csv`의 주택용 총사용량과 한전 가구 평균 자료의 cross-check

개별 고객/가구 상세 데이터를 받아야 가능한 분석:

- 15분 단위 load shape clustering
- 낮 시간대 재생에너지 흡수 가능 수요 찾기
- DR 또는 요금제 반응 분석
- 가구별 피크 시간, 냉난방 패턴, EV/태양광/ESS 추정

## 다음 액션

1. 한전 전력데이터 개방포털에 로그인해서 `가구 평균 전력 사용량`을 시군구/월별로 다운로드한다.
2. 동시에 공공데이터포털 `가구 평균 전력사용량` OpenAPI 활용신청을 해둔다.
3. 개별 고객 15분 데이터가 필요하면 EDS 사업자용 API 절차와 고객 동의 절차를 확인한다.
4. MVP 분석에는 먼저 공개 평균 자료를 쓰고, 고객 단위 분석은 파일 다운로드가 아니라 제휴/동의 기반 데이터 수집으로 설계한다.
