# Processed 데이터 상관관계 및 가설 검증

## 1. 분석 범위

- 사용 데이터: `load_hourly_all_sources`, `smp_hourly`, `weather_daily`, `supply_daily`, `dr_hourly`
- 분석 구간: 주로 `2020~2025` (SMP/공급 데이터 가용 구간)
- 분석 표본(시간단위): 114,216 rows (date-hour 중복 제거 후)

## 2. 가설 및 검증 결과

### H1. 주말 낮(11~15시) 수요는 평일 낮보다 낮다
- 평일 평균: 69,037.3 MW
- 주말 평균: 55,269.5 MW
- Welch t-test: t=77.895, p=< 0.0001
- 판정: 채택

### H2. 수요(load_mw)와 SMP는 양(+)의 상관관계를 가진다
- Pearson r: 0.336 (p=< 0.0001)
- Spearman rho: 0.374 (p=< 0.0001)
- 판정: 채택

### H3. 기온-수요 관계는 U-shape(2차식)가 1차식보다 설명력이 높다
- Linear R²: 0.0037
- Quadratic R²: 0.2589
- 2차항 계수(a): 53.4135
- 판정: 채택

### H4. DR 활성 시간대(입찰>0)는 비활성 시간대보다 SMP가 높다 (2020~2021)
- DR 활성 평균 SMP: 82.37
- DR 비활성 평균 SMP: 76.00
- Welch t-test: t=18.316, p=< 0.0001
- DR 활성 시간 수: 11,976
- 판정: 채택

### H5. SMP=0 시간대는 주말 낮(11~15시)에 집중된다
- 교차표(weekend_midday x zero_smp):

```text
zero_smp        False  True 
weekend_midday              
False           49694      4
True             3129     21
```
- Chi-square: chi2=258.007, p=< 0.0001
- 판정: 채택

### H6. 저부하일의 예비율은 고부하일보다 높다
- 저부하(하위 25%) 평균 예비율: 42.35%
- 고부하(상위 25%) 평균 예비율: 19.80%
- Welch t-test: t=40.634, p=< 0.0001
- 판정: 채택

## 3. 종합 해석

- 계통 스트레스는 '주말 낮 저부하 + 일부 시간의 가격 급락(SMP=0)' 패턴으로 통계적으로 확인된다.
- 수요와 SMP는 유의한 양의 상관관계를 보여, 저수요 시간대의 가격 약세 가설을 지지한다.
- 온도-수요 관계는 단순 선형보다 비선형(U-shape) 설명력이 높아, 계절/기온 기반 운영 전략이 필요하다.
- DR 활성은 고가격 구간과 연관되지만, 인과관계라기보다는 '동시 반응' 가능성이 크다.
