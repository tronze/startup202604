# Documentation Index

문서는 아래처럼 정리했다.

## Current

현재 프로젝트에서 바로 쓰는 문서다.

- `current/data_reference/`
  - 데이터 인벤토리, 외부 데이터 확인, GIS/PVOUT 관련 정리
- `current/research/`
  - 검증 메모, 연구노트, 분석 결과

## Legacy

이전 문제정의, 주장 검증, 수집전략, 대외 요청 템플릿이다.  
현재 프로젝트와 직접 섞지 않기 위해 분리했다.

- `legacy/`
  - 사업 아이디어/주장 문서
  - 해외 벤치마크
  - 추가 수집 데이터 리스트
  - 링크 모음
  - 기관협조 메일 템플릿

## Shared Writing Rule

공유 대상 문서에는 아래 원칙을 적용한다.

- 절대 경로(`/Users/...`)를 쓰지 않는다.
- 로컬 파일 링크 대신 저장소 상대경로를 텍스트로 적는다.
- 외부 출처는 URL을 직접 적는다.
