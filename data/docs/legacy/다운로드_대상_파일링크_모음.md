# 다운로드 대상 파일-링크 모음 (확장판 정렬본)

## 목적

`주장증명을_위한_추가수집_데이터리스트.md`의 전체 범위를 실제 수집 가능한 링크/요청 대상으로 1:1 매핑한 문서다.

핵심 원칙:

- 모든 주장별 필수 데이터 항목을 누락 없이 링크화
- 수집 방식은 `직접다운로드/API` → `Playwright` → `기관협조` 순으로 실행
- 링크 문서와 `data/raw/download_targets_manifest.csv`를 동일 범위로 유지

## 5) 운영 규칙 (배치 수집)

1. `download_targets_manifest.csv`의 `priority=1`부터 일괄 수집  
2. `access_type=direct_file/api`를 먼저 실행  
3. `playwright_needed=yes`만 별도 큐로 브라우저 자동화  
4. `access_type=institution_request`는 요청서 템플릿으로 병렬 진행  
5. 실행 후 `download_ready`를 `no/partial/yes`로 갱신
